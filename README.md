### Quick Start with Auto-Updater

1. **Download the updater**: Get `RobloxScriptUpdater.exe` from the releases page
2. **Place it with the script**: Put the .exe in the same folder as `ESPAIMBOTWALLBANGROBLOX.lua`
3. **Run the updater**: Double-click `RobloxScriptUpdater.exe`
4. **Click "Check for Updates"**: The updater will handle everything!

### Building the Updater Yourself

See [UpdateChecker/README.md](UpdateChecker/README.md) for detailed build instructions.

**Windows:**
```bash
cd UpdateChecker
build.bat
```

**Linux/Mac:**
```bash
cd UpdateChecker
./build.sh
```

The compiled `.exe` will be in `UpdateChecker/bin/Release/net6.0-windows/win-x64/publish/`
